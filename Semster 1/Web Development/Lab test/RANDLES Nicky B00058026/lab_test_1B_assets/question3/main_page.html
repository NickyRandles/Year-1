Expert web: Firefox "add-ons"

What are Firefox add-ons and why install them?

Firefox add-ons (a.k.a.Extensions) are extra features that you can install into your web
browser application. One good example is HttpFox , which lets you record and view the
details of the HTTP request messages from Browser to server, and the incoming HTTP
response messages from server back to browers. Others can be found on the featured
page.
